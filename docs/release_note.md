# Release Note

## 2026-03-12

- Standardized the public-facing product name to `Byte` while keeping the `byte` package name stable.
- Tightened safe semantic defaults to reduce false positives in production.
- Added `/metrics`, `/healthz`, and `/readyz` to `byte_server`.
- Added explicit origin, semantic-risk, observability, and deployment disclosures across the top-level docs.

## Historical Note

Earlier release notes in this repository referred to the legacy `Byte` product
name and predated the current documentation cleanup. They should not be used as
the primary source for current positioning or enterprise readiness claims.
